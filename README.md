## Getting Started

Run the following commands to clone and run it.

### To clone the repository

  `git clone git@github.com:Amtresu/Leaderboard.git`

### To enter Directory

`cd Leaderboard`

### To Install Packages

`npm install`

### To run project on local server

`npm run server`

### To open project with vs code 

`code .`
